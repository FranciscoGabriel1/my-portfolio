# Inspirations

Add screenshot images here to use as design references.

Accepted formats: `.png`, `.jpg`, `.webp`

Examples of what to include:
- Screenshots of portfolio sites you like
- Design system references (Apple, Google, Linear, Vercel)
- Color palette screenshots
- Layout references (hero sections, card styles, navbars)

> The AI will read this folder when building the project to align with your visual references.
